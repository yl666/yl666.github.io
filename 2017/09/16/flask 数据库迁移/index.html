#使用命令创建init迁移仓库 python manage.py db init

使用migrate命令创建迁移脚本 python manage.py db migrate -m ‘add user filed to Usertable’

###上面第二条命令的作用：在执行成功之后，会在/migrations/version的文件夹下面查看，迁移脚本中的upgrade()和downgrade函数

它的作用分别是：
upgrade():将迁移中的改动应用到数据库中
downgrade():将迁移中的改动从数据库里面删除，也就是说具有回滚到某个迁移点的功能

使用Upgrade命令将迁移中的改动应用到数据库中
python manage.py db upgrade 如果是是在首次创建的话，相当于db.create_all()

使用downgrade命令可以会滚迁移数据库中的改动
